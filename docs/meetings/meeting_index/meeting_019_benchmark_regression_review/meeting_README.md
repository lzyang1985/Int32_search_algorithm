## 关键上下文

### @Host 测试环境特殊性
- 瓶颈在**内存**
- 对硬件压榨很狠
- 关闭所有不必要软件后测试
- **任何额外操作都会导致极大偏差**
- 这意味着 meeting_018 的环境可能不够"干净"

### meeting_018 残余代码变更
- D-140: 2x SIMD 循环展开 → ⚠️ `#ifdef` 条件编译默认关闭
- D-141: 32B 对齐分配 → ✅ 保留
- D-142: 小桶 (<8) 标量快速路径 → ✅ 保留
- D-143: 防御性边界检查 → ✅ 保留（加固版）

### meeting_017 上下文
- B1 路径 ~88% 时间花在内存子系统
- 理论下界：微架构 ~310cy，纯算法 ~90cy
- "内存瓶颈"是项目的核心挑战
